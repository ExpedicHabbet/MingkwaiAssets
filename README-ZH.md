## 什么是明快打字机？

明快打字机由林语堂于1947年5月22日发明。长14英寸（36厘米），宽18英寸（46厘米），高9英寸（23厘米），稍大于普遍的打字机。使用巨大的滚筒存贮数以万计的汉字、电动机驱动、只看字顶部和底部的独特检字方法并透过屏幕（或“魔眼”）确认。可惜的是，由于其精巧的结构，未能实现量产。现在差评君等正在复刻中。
